# Log Viewer Requirements

Applies to all log view pages (currently: GitLab job log viewer).

1. **Full page width** — use the entire viewport width. Do not constrain with max-width.

2. **Autolink URLs** — any bare URL in log output (http/https) must be rendered as a clickable link.

3. **Sensible font size** — use a compact, readable size. Not the large defaults fashionable in modern UIs. Target 12px for log content, 14px for page chrome, and proportionally small headings.
